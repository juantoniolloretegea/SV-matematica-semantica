Función de esta pieza audiovisual

El vídeo no sustituye el Documento III, sino que actúa como apoyo explicativo para facilitar su comprensión. Su lugar natural es esta página auxiliar, enlazada desde el documento doctrinal y servida desde la misma infraestructura web del repositorio.

  • mantiene la referencia doctrinal principal en la Release 2 de PubPub;
  • preserva el control material del activo audiovisual dentro del propio repositorio;
  • permite una reproducción directa sin recurrir a plataformas externas.
