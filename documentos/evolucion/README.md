# Álgebra de evolución discreta del Sistema Vectorial SV

**Subtítulo:** Frame, factor externo, cascada y trayectoria auditable
**Estado:** Esqueleto conceptual v0.3.0 — pendiente de formalización

---

## Función en el repositorio

Este directorio albergará la formalización del plano temporal del Sistema Vectorial SV: qué ocurre cuando un sistema de células acopladas se evalúa más de una vez a lo largo del tiempo, y cómo se registra, se estructura y se audita esa secuencia de evaluaciones.

## Relación con los demás documentos

- Los **Fundamentos** definen la célula y el motor normativo.
- La **Composición** (I y II) define cómo las células se acoplan en un instante.
- La **Evolución discreta** define qué pasa cuando el tiempo avanza.

Son planos ortogonales: la composición es espacial (dentro de un frame); la evolución es temporal (entre frames).

## Conceptos principales del esqueleto

- **Frame** como unidad temporal inmutable del sistema.
- **Factor ν** como registro formal de lo que ocurrió entre dos frames.
- Distinción **ν exógeno** (el mundo actuó) / **ν endógeno** (el sistema motivó una acción).
- **Cascadas de reacción** a través de la composición espacial.
- **Trayectoria** T = (S₁, ν₁, S₂, ν₂, …, Sₙ) como película auditable.
- Tres horizontes graduados: trayectoria clínica, sistema reactivo, humanoide.

## Estado actual

El esqueleto conceptual v0.3.0 existe como documento de debate. No ha pasado revisión adversarial formal. Su incorporación a este repositorio como documento doctrinal requerirá el mismo proceso de revisión que los demás documentos de la serie.
