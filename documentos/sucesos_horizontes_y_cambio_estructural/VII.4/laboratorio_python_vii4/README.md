## Alcance

El laboratorio verifica e ilustra, sobre la célula SV(9,3), cinco piezas del documento:

1. respuesta estructural;
2. persistencia de transporte;
3. umbral de ruptura y transición controlada;
4. siembra doctrinal de células especializadas;
5. corolarios operativos iniciales, uno por cada familia especializada.

No constituye backend soberano, IR, validator ni integración cerrada con el Lenguaje SV.
