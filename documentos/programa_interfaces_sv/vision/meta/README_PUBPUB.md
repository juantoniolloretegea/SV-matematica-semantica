# Relación con PubPub

## Finalidad

Este archivo fija la relación entre los materiales del carril de visión conservados en el repositorio y la publicación efectivamente difundida en PubPub.

## Piezas relevantes

### Manuscrito fuente

- `../manuscrito/paper_sv_vision_pubpub.md`

Es el manuscrito base en Markdown, conservado en el repositorio con enlaces relativos a `../figuras/`.

### Versión publicada en HTML

- `../manuscrito/paper_sv_vision_publicado_pubpub.html`

Es la copia HTML exportada de la publicación ya difundida, conservada como referencia estable de la forma efectivamente publicada.

## Criterio de uso

- El archivo `.md` es la base editable y trazable del manuscrito.
- El archivo `.html` conserva la versión publicada y sirve como referencia pública de contraste.
- Ninguno de estos archivos sustituye a las figuras del carril, que permanecen en `../figuras/`.

## Observación

El HTML exportado por PubPub puede incluir elementos técnicos propios del sistema de exportación o de librerías de renderizado. Eso no altera la autoría ni la licencia pública del trabajo, pero aconseja distinguir entre:

1. manuscrito fuente;
2. HTML publicado exportado;
3. y materiales auxiliares del repositorio.
