# Manuscrito

## Finalidad

Este directorio contiene el manuscrito listo para PubPub y la copia HTML publicada de referencia del carril de visión del Sistema Vectorial SV.

## Contenido

- `paper_sv_vision_pubpub.md`  
  Manuscrito fuente en Markdown, con referencias relativas a `../figuras/`.

- `paper_sv_vision_publicado_pubpub.html`  
  Copia del HTML publicado en PubPub como referencia pública estable del carril de visión.

## Criterio de uso

- El archivo `.md` es la base editable del manuscrito.
- El archivo `.html` conserva la forma efectivamente publicada.
- Las figuras del carril viven fuera de este directorio, en `../figuras/`.
