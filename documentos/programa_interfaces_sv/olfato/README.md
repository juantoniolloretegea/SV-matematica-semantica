## Finalidad

Este paquete reúne el manuscrito, las figuras, el laboratorio demostrativo mínimo y los metadatos editoriales del carril olfativo del Programa de interfaces del Sistema Vectorial SV.

## Contenido

- `manuscrito/`: manuscrito principal para publicación.
- `figuras/`: portada y figuras del documento.
- `laboratorio/`: protocolo, casos, matriz de refutación y laboratorio mínimo en Python.
- `meta/`: referencias, manifiesto de activos y notas documentales.

## Estatuto de la pieza

La propuesta no constituye todavía una API, un módulo paralelo ni una integración técnica inmediata al SV. Su función es clarificar, poner a prueba y documentar una primera formulación formal de admisibilidad olfativa dentro del ecosistema SV.
