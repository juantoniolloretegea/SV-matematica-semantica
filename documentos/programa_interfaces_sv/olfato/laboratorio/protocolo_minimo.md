# Protocolo mínimo de aplicación del modelo

## Unidad de análisis
La unidad mínima es un **episodio de comparecencia quimiosensorial** descrito de forma acotada.

## Secuencia de evaluación
1. **Comparecencia**: descripción del episodio sin clausura previa.
2. **Captura**: cumplimentación de la ficha mínima.
3. **Admisibilidad**: decisión entre admisibilidad suficiente, admisibilidad con `U`, inadmisibilidad o reapertura.
4. **Cierre**: solo si la admisibilidad lo permite.

## Ficha mínima de captura
- vía de acceso: ortonasal / retronasal / indeterminada
- contexto: ambiental / ingestivo / mixto
- evidencia de componente volátil: suficiente / insuficiente / contradictoria
- coactivación gustativa: presente / ausente / no separable
- coactivación chemestésica: presente / ausente / no separable
- carga textural o térmica: relevante / no relevante / indeterminada

## Regla mínima para determinar la vía
- **Ortonasal**: comparecencia relevante sin ingestión y con exposición ambiental o aproximación externa suficiente.
- **Retronasal**: comparecencia relevante durante masticación, deglución o persistencia aromática post-ingestiva.
- **Indeterminada**: no se pueden separar ambas vías con base suficiente o concurren simultáneamente sin desagregación razonable.

## Regla mínima para evidencia volátil suficiente
Se entenderá como base explícita no arbitraria para tratar el episodio como odorífero:
- presencia reconocible de componente aromático por vía aérea o post-ingestiva;
- coherencia entre contexto, vía presunta y descripción del episodio;
- ausencia de contradicción fuerte con los demás campos de captura.

## Regla de prudencia
Ante duda entre cierre fuerte, `U` o inadmisibilidad, el modelo preferirá **`U` o inadmisibilidad** antes que una adscripción modal forzada.
