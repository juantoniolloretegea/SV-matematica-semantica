# Notas GitHub y relación con el programa

## Relación con el Programa de interfaces
Si se mantiene la convención observada en el repositorio público del programa, los carriles previos se localizan bajo:

- `documentos/programa_interfaces_sv/semantica/`
- `documentos/programa_interfaces_sv/vision/`
- `documentos/programa_interfaces_sv/motricidad/`

Este paquete está preparado para que el carril nuevo se integre, cuando corresponda, de forma hermana bajo una ruta análoga:

- `documentos/programa_interfaces_sv/olfato/`

## Cautela
La ruta de `olfato/` debe entenderse como **propuesta estructural coherente** con el patrón actual del programa. La integración efectiva debe verificarse en el momento de la subida.
