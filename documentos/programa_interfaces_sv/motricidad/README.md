# Carril de motricidad del Sistema Vectorial SV

Este directorio contiene el carril de motricidad del Sistema Vectorial SV, centrado en una condición estructural de frontera entre trayectoria interna y posible exposición externa.

## Estructura

- `manuscrito/`  
  Manuscrito fuente del trabajo.

- `figuras/`  
  Figuras SVG citadas por el manuscrito.

- `codigo/`  
  Evaluador verificable y dataset de casos.

- `meta/`  
  Material auxiliar de control editorial, documental y bibliográfico.

## Alcance de esta fase

Este carril no formaliza actuadores, cinemática ni control físico. Su alcance es más austero: fijar una condición estructural de exposición previa a cualquier acoplamiento externo.
