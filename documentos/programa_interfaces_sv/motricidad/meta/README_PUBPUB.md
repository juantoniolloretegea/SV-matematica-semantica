# Relación con PubPub

## Estado actual

En el material recibido para este carril no consta todavía una copia HTML publicada en PubPub.

## Criterio

Cuando exista publicación en PubPub, este directorio deberá conservar:

- el manuscrito fuente en Markdown;
- la copia HTML publicada;
- y la trazabilidad mínima entre ambos.

Mientras tanto, el carril conserva el manuscrito fuente y el material auxiliar necesario para su futura publicación.
