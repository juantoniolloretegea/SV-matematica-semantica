## Función en el repositorio

Este documento es la verdad normativa suprema del Sistema Vectorial SV. Todo lo que este repositorio contiene — y todo lo que los repositorios SVcustos-dataset y SVperitus-dataset implementan — queda subordinado a lo que este documento establece.

## Contenido

Fija la formulación matemática y arquitectónica del Sistema Vectorial SV:

- La célula SV exacta como producto cartesiano finito de estados ternarios.
- La representación polar cerrada.
- El motor normativo determinista con umbral T(n) = ⌊7n/9⌋.
- El estatuto epistémico de U como estado de no determinación actual.
- La álgebra semántica tipada para composición multicelular.
- La familia de operadores ℱ_SV = { max, min, ⊗_gate, σ_{k,φ}, ▷, Γ, Comp }.
- La subordinación de la inteligencia artificial al marco.
- Los invariantes constitutivos del sistema.
