# Tests locales

Este directorio contiene tests de humo y validación básica del entorno local.

## Archivos

- `test_runner.py`
- `test_no_clausura.py`
- `test_labs_smoke.py`

## Función

No sustituyen una batería extensa, pero sirven como compuerta mínima para comprobar que:

- el runner responde,
- la no clausura se mantiene,
- y los laboratorios locales cargan y ejecutan sin ruptura inmediata.
