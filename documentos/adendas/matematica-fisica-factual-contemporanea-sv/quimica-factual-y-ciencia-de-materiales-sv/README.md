# Química factual y ciencia de materiales en el Sistema Vectorial SV

Este directorio ordena los desarrollos de química factual, tabla periódica estructural, candidatos SV, pares estructurales, enlace, aleación y ciencia de materiales dentro del ámbito de matemática y física factual contemporánea del Sistema Vectorial SV.

La carpeta reúne tres publicaciones principales actualmente activas:

1. [`genesis-del-hidrogeno/`](genesis-del-hidrogeno/) — **Génesis del hidrógeno y teoría de la persistencia energética estructural**.  
   Fija el hidrógeno como caso canónico conocido de persistencia atómica discreta estable, desarrolla la tabla periódica estructural inicial y formula el criterio operatorio de persistencia, frontera, residual e identidad compatible.

2. [`elementos-materiales-nueva-generacion/`](elementos-materiales-nueva-generacion/) — **Análisis preliminar de elementos químicos, materiales y aleaciones de nueva generación para usos médicos y científicos**.  
   Extiende el dominio hacia el catálogo SV-443: los primeros 118 elementos corresponden al dominio reconocido, y los elementos `k = 119…443` se tratan como candidatos estructurales SV no detectados empíricamente, con estatuto operatorio `U`.

3. [`catalogo-pares-estructurales/`](catalogo-pares-estructurales/) — **Catálogo de Pares Estructurales SV (CPS-SV): enlace, aleación y compatibilidad posicional desde los 118 elementos base hasta los 443 candidatos del dominio extendido**.  
   Desarrolla el dominio de pares estructurales sobre `Ω₄₄₃`, aplica criterios de admisibilidad de enlace y aleación, y genera el catálogo completo de pares no ordenados del dominio SV-443.

---

## Función del directorio

Este directorio no es una colección ornamental de publicaciones independientes. Su función es ordenar una secuencia material de química factual:

```text
génesis estructural del hidrógeno
→ tabla periódica estructural
→ candidatos SV-443
→ pares estructurales
→ enlace, aleación y compatibilidad posicional
→ ciencia de materiales SV
```

La lectura correcta distingue tres planos:

- **dominio reconocido**: los 118 elementos de la tabla periódica actualmente aceptada;
- **dominio estructural extendido**: candidatos SV no detectados empíricamente, conservados bajo estatuto operatorio `U`;
- **dominio de pares**: combinaciones estructurales admisibles o no admisibles entre elementos del dominio completo `Ω₄₄₃`.

- **Tabla 1**: subdominio extendido `Ω_ext = {119,…,443}`;
- **Tabla Global**: dominio completo `Ω₄₄₃ = Ω₁₁₈ ∪ Ω_ext`.

## Advertencia de alcance

Las magnitudes tabuladas en estos trabajos son magnitudes estructurales SV. No deben confundirse con mediciones empíricas ya verificadas de nuevos núcleos atómicos, propiedades físico-químicas experimentales o síntesis material realizada.

Los candidatos estructurales no detectados empíricamente conservan estatuto `U` hasta contraste externo suficiente. La función del corpus SV en este directorio es generar, ordenar y someter a falsación un dominio estructural de búsqueda, no declarar como experimentalmente realizados los candidatos no observados.
