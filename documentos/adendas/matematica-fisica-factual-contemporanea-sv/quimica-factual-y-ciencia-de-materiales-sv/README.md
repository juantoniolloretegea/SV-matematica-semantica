# Química factual y ciencia de materiales en el Sistema Vectorial SV

Este directorio ordena los desarrollos de química factual y ciencia de materiales dentro del ámbito de matemática y física factual contemporánea del Sistema Vectorial SV.

La primera publicación activa es:

- `genesis-del-hidrogeno/` — Génesis factual del hidrógeno, tabla periódica estructural y duplicación periódica SV.

Este directorio raíz no abre simultáneamente todos los desarrollos derivados. Registra una sede de ordenación para trabajos posteriores sobre aleaciones estructurales, superconductividad estructural, materiales bidimensionales, materiales aeroespaciales, materiales médicos, radiactividad, residuos, costes, normalización y ciencia de materiales.
